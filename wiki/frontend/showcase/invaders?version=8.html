Rocket Invaders from Mars

Rocket Invaders from Mars is the sample application distributed with libRocket. It comes with complete source code and runs on Windows, MacOSX and Linux. The game contains original '70s-style game play and graphics with a modern HTML/CSS style user interface.

A pre-built version is available for Windows.
